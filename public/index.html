LUBDUB Share

Share files without internet, without a router, and without cables by creating a temporary Wi-Fi Direct session between nearby PCs.

This Device

Host Session

This device becomes the session owner and exposes an invite code containing the Wi-Fi Direct SSID and passphrase. You can also send a connect request from the nearby device list and let the other PC approve it.

Join Session

Paste the invite code from the host device. The app will connect to the Wi-Fi Direct network, find the host, and register this PC. If the other device sends a connection request, approving it below will connect automatically.

Nearby Devices

Devices on the same local network can receive a ShareIt-style connection request from here.

Connection Requests

When another device asks to connect, approve it here and this PC will join automatically.
